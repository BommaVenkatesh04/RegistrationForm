Attribute in HTML!

HTML attributes are special words used within the start tag of an element to provide additional information, configure its behavior, or adjust its display. They always come in name/value pairs, separated by an equal sign (=), with the value enclosed in quotation marks (e.g., name="value").

The contenteditable attribute is a global HTML attribute that makes the content of any HTML element editable by the user. When applied, the element behaves like a rich text editor or an input field directly on the webpage.
